### Change Personal Information
Edit the HTML file to include:
- Your name and professional title

### Modify Colors
Update CSS custom properties:
```css
:root {
  --primary-color: #2563eb;
  --secondary-color: #64748b;
  --accent-color: #f59e0b;
  --text-dark: #1e293b;

  --bg-light: #f8fafc;
}
```
